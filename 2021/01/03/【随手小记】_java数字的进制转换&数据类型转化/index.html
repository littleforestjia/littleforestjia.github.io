java数字的进制转换&数据类型转化

进制转换

  1. 十进制转成二进制

    String s = Integer.toBinaryString(Integer n)：将十进制整型数转成二进制字符串。

  2. 十进制转成十六进制

    String s = Integer.toHexString(Integer n)： 将十进制整型数转成十六进制字符串。

数据类型转化

  1. 整型转成字符串

    String s = String.valueof(Integer n)：将整型数字转成字符串。

  2. 整型转成字符串

    String s = Integer.toString(Integer n)：将整型数字转成字符串。

  3. 字符串转成整型

    Integer a = Integer.valueof(String s)：将字符串转成整型数字。

    Integer a = Integer.parseInt(String s)：将字符串转成整型数字。

  4. 字符数组转成字符串

    String s = new String(Char[] chars)：使用String的构造函数将字符数组转成字符串。

  5. 字符串转成字符数组

    String s = “abcdsfkk”;

    char[] chars = s.toCharArray();

    使用toCharArray()方法将字符串转成字符数组。
